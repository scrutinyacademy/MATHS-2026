Find probability of getting a vowel from "EDUCATION"
Vowels = E, U, A, I, O = 5
Total letters = 9
P = Favourable / Total
P = 5/9
Q32 MARKS
Express tan θ in terms of sin θ
tan θ = sin θ / cos θ
cos θ = √(1 – sin²θ)
∴ tan θ = sin θ / √(1 – sin²θ)
Q42 MARKS
Draw diagram for tower with angle of elevation 45°
Draw right triangle with:
– Angle of elevation = 45° at base
– tan 45° = h/d = 1
∴ height = distance
Q52 MARKS
Check if triangle with sides 2√2, 4, 2√6 is right-angled
(2√6)² = 24
(2√2)² + 4² = 8 + 16 = 24
Since (largest)² = sum of squares of other two
✔ Yes, it is a right-angled triangle
Q62 MARKS
Solve: 2sin²θ – 3sinθ + 1 = 0
Let x = sin θ  →  2x² – 3x + 1 = 0
(2x – 1)(x – 1) = 0
x = 1/2 or x = 1
θ = 30° or θ = 90°
📝 SECTION – II (4 Marks)
Q74 MARKS
Write and explain the formula of Mode for grouped data
Mode = l + [ (f₁ – f₀) / (2f₁ – f₀ – f₂) ] × h

l = lower boundary of modal class
f₁ = frequency of modal class
f₀ = frequency of class before modal class
f₂ = frequency of class after modal class
h = class width
Q84 MARKS
Prove: (1 + tan²A) / (1 + cot²A) = tan²A
LHS = (1 + tan²A) / (1 + cot²A)
= sec²A / cosec²A
= (1/cos²A) / (1/sin²A)
= sin²A / cos²A
= tan²A = RHS ✔
Q94 MARKS
The area of a rectangle is 77 sq.cm. Length exceeds breadth by 4 cm. Find dimensions.
Let breadth = x  →  length = x + 4
x(x + 4) = 77
x² + 4x – 77 = 0
(x + 11)(x – 7) = 0
x = 7 (taking positive value)
Breadth = 7 cm, Length = 11 cm
Q104 MARKS
Prove: n(A∪B) = n(A) + n(B) – n(A∩B)
n(A∪B) = n(A) + n(B) – n(A∩B)

Proof: A∪B = A ∪ (B – A) [disjoint sets]
∴ n(A∪B) = n(A) + n(B–A)
n(B–A) = n(B) – n(A∩B)
∴ n(A∪B) = n(A) + n(B) – n(A∩B) ✔
Q114 MARKS
AP: a = 1, last term l = 20, sum S = 399. Find number of terms n.
S = n/2 × (a + l)
399 = n/2 × (1 + 20)
399 = n/2 × 21
n = 399 × 2 / 21
n = 38
📝 SECTION – III (6 Marks)
Q126 MARKS
Draw graph of y = x² + 2x – 3 and find zeroes
Step 1: Factorise → (x+3)(x–1) = 0
Zeroes: x = –3 and x = 1

Step 2: Table of values
x = –3 → y = 9 – 6 – 3 = 0
x = –2 → y = 4 – 4 – 3 = –3
x = –1 → y = 1 – 2 – 3 = –4
x =  0 → y = 0 + 0 – 3 = –3
x =  1 → y = 1 + 2 – 3 = 0

Step 3: Plot points, draw smooth parabola
Zeroes = –3 and 1
Q136 MARKS
Heights & Distances: Tower at 45° elevation, distance = 10m. Find height.
Draw right triangle with angle 45° at base
tan 45° = h/d
1 = h/10
Height = 10 m
Q146 MARKS
Prove 2√3 + √5 is irrational
Assume 2√3 + √5 is rational = p/q
Then √5 = p/q – 2√3
Squaring → leads to √3 being rational
But √3 is irrational → contradiction!
∴ 2√3 + √5 is irrational ✔

🧠 Marking Strategy

How examiners give marks · Follow this = full marks

🟢 2 Mark Questions

  • 1 mark → Formula / method
  • 1 mark → Final answer
  • Always write formula first
  • Write units if needed
  • Short but complete steps

🟡 4 Mark Questions

  • 1 mark → Formula
  • 2 marks → Steps
  • 1 mark → Final answer
  • Show LHS → RHS for proofs
  • Don't skip middle steps

🔴 6 Mark Questions

  • 1 mark → Formula
  • 3–4 marks → Steps
  • 1 mark → Diagram/Graph
  • 1 mark → Final answer
  • Neat diagram mandatory

⚠️ Common Mistakes (Big Mark Losses)

  • Writing answer without formula
  • No diagram in graph questions
  • No steps in construction
  • Skipping units (cm, m, cm², m²)
  • Direct answer without showing work
  • Incomplete LHS → RHS in proofs
  • No table in graph questions
  • Not labelling axes in graphs

🔥 Golden Rules (Follow This = 80/80)

  • Always write: Given → Formula → Solution → Answer
  • Use proper steps — never skip
  • Draw neat, labelled diagrams
  • Underline or box the final answer
  • Write units clearly (cm, cm², m, etc.)
  • Even if answer is wrong → correct steps = 2–4 marks
  • Correct answer without steps = 0–1 mark only!

🎯 Section-wise Specific Tips

📈 Graph Questions (Strict Marking)
1. Write the equation clearly
2. Make a table with at least 3–4 values of x and y
3. Plot all points accurately
4. Draw a smooth parabola
5. Mark and write the zeroes on x-axis
No table = marks cut · No labels = marks cut
📏 Construction Questions
Must write numbered steps: 1. Draw BC = 7 cm · 2. Draw angle 45° at B ...
Steps must match the diagram
No written steps = 0 marks even if diagram is correct!
📊 Statistics (Mode)
Write full formula first · Identify modal class · Substitute values
Direct answer without formula = marks cut
📦 Surface Areas & Volumes
Write individual formulas → substitute → add/subtract
Always write units: cm², m², cm³
Missing units = –0.5 to –1 mark

🟡 4 Mark Example — AP Problem

Given: a = 1, l = 20, S = 399

Formula:
S = n/2 × (a + l)

Solution:
399 = n/2 × (1 + 20)
399 = n/2 × 21
n = (399 × 2) / 21 = 798 / 21

Answer: n = 38 terms ✔

📌 Arithmetic Progression

aₙ = a + (n–1)d

Sₙ = n/2 × (2a + (n–1)d)

Sₙ = n/2 × (a + l)

📌 Probability

P(E) = Favourable outcomes
         ─────────────────
         Total outcomes

0 ≤ P(E) ≤ 1

📌 Trigonometry

sin²θ + cos²θ = 1

1 + tan²θ = sec²θ

1 + cot²θ = cosec²θ

tanθ = sinθ / cosθ

📌 Statistics

Mode = l + (f₁–f₀)/(2f₁–f₀–f₂) × h

Mean = Σfx / Σf

Median = l + (n/2–cf)/f × h

📌 Coordinate Geometry

Distance = √((x₂–x₁)²+(y₂–y₁)²)

Centroid = ((x₁+x₂+x₃)/3, (y₁+y₂+y₃)/3)

Section = ((mx₂+nx₁)/(m+n), (my₂+ny₁)/(m+n))

📌 Surface Areas

Cylinder TSA = 2πr(h+r)
Cylinder Volume = πr²h

Cone CSA = πrl
Cone Volume = (1/3)πr²h

Sphere SA = 4πr²
Hemi SA = 3πr²

📌 Heights & Distances

tan 30° = 1/√3
tan 45° = 1
tan 60° = √3

sin 30° = 1/2, cos 30° = √3/2
sin 45° = 1/√2, cos 45° = 1/√2
sin 60° = √3/2, cos 60° = 1/2

📌 Sets & Venn Diagrams

n(A∪B) = n(A)+n(B)–n(A∩B)

A–B = A ∩ B'

n(A) = n(A–B) + n(A∩B)

n(U) = n(A)+n(B)–n(A∩B)+n((A∪B)')
